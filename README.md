# NGO Website Feature Highlight – Task 5

This project was created as part of the InAmigos Foundation internship program.

The design highlights essential features that should be present in an NGO website:

* Donation Section
* Volunteer Registration
* Impact Showcase
* Testimonials
* Campaigns & Events
* Contact & Support

Files Included:

* HTML Design File
* PDF Export

Objective:
To understand and present important NGO website features that improve user engagement, transparency, volunteering, donations, and communication.
